# 🤖 Agentic RAG System

A production-ready Agentic Retrieval-Augmented Generation system. Upload PDFs and ask questions with intelligent retrieval, multi-step reasoning, and self-reflection.

## Setup

Add your API key under **Settings → Variables and secrets**:

- `GOOGLE_API_KEY` — Google Gemini (recommended, free tier)
- `OPENAI_API_KEY` — OpenAI GPT (alternative)

Then the Space will automatically start the backend and the Streamlit UI.
